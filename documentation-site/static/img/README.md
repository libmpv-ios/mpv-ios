# Image assets

- `logo.png`, `favicon.ico`, `social-card.png` — all derived from the
  same source as the iOS app's own icon: mpv-android's launcher icon
  (see the main repo's `mpv-ios-player/Assets.xcassets/AppLogo.imageset/`
  for the original, and the main README's "App icon & logo" section for
  the full provenance/licensing note).
- These were generated programmatically (resize + recolor background)
  rather than hand-designed. If you'd like to contribute a
  professionally redesigned logo/favicon, see the main repo's
  `CONTRIBUTING.md` — a native, high-resolution icon is explicitly listed
  as a welcome contribution area, and updating these site assets to match
  would be a natural follow-up to that.
